Spring Boot Homework Project

Welcome to the MyLibrary project! This is a simple web application developed using Spring Boot. The purpose of this project is to manage authors and books in an intuitive and simple way.

Project Goal

  • Create a web application to manage authors and books.
  • Implement functionalities like adding, viewing, editing, and deleting books and authors.

Key Features

  • A form to add new authors and books.
  • Display a list of all books and authors.
  • Detailed view of books and authors.
  • Edit or delete authors and books.

Technologies Used

  • Spring Boot
  • H2 Database
  • Thymeleaf for rendering views
  • FontAwesome for icons
